
# Premium Neon Button Template

Interactive, mobile-ready, and fully responsive neon button template. Perfect for landing pages, hero sections, or any modern web project.

## ✅ Features
- Glowing neon buttons with hover and click effects
- Interactive sparks follow mouse or touch
- Fully responsive on mobile and desktop
- Lightweight HTML, CSS, and optional JS
- Easy to customize colors, sizes, and effects

## 📂 Folder Structure

## 🎨 Customization
- Change neon color: update `--neon-color` in `style.css`
- Adjust button size: modify padding and font-size
- Change spark behavior: edit `sparks-canvas` JS in `index.html`
